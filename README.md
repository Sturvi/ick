# ICK Landing Page

This is a static, responsive landing page for Investment Company of Karabakh (ICK).

Contents:
- index.html — Semantic HTML (Home, About, Services, Gallery, Partners, Contacts)
- assets/css/styles.css — Styles (variables, responsive, accessible)
- assets/js/main.js — Small JS (mobile nav, section highlight)
- logo/ick-latest.svg — Used in header and favicon

How to edit:
- Replace placeholder texts and add real images/logos
- Update contact details when finalized

Hosting:
- Works on any static host (e.g. Netlify, Vercel, GitHub Pages, Nginx)

Notes:
- no external dependencies, includes basic SEO/OG tags
