# Анализ цветов логотипа ICK

## Исходный логотип (logo/ick-latest.svg)

### Цветовая палитра логотипа:
- **Основной цвет**: `#ffffff` (белый)
- **Применение**: Все графические элементы и текст

### Особенности:
- Логотип полностью **монохромный** (белый)
- Предназначен для использования на **темном фоне**
- Современный минималистичный дизайн

## Текущая цветовая схема сайта

### Hero Section (index.html:47-59)
```css
background: linear-gradient(135deg, #1e3c72 0%, #2c5f8d 50%, #6fa3d9 100%);
```

**Цвета градиента**:
- `#1e3c72` - Темно-синий (темнее)
- `#2c5f8d` - Синий средний (основной корпоративный)
- `#6fa3d9` - Голубой светлый

### Primary Color (assets/css/site.css)
```css
--primary-color: #2c5f8d;
```

## Рекомендации по обновлению цветовой схемы

### Вариант 1: Сохранить синюю гамму (рекомендуется)

Так как логотип белый и уже хорошо смотрится на синем фоне, можно:

1. **Сделать градиент более насыщенным и глубоким**:
   ```css
   background: linear-gradient(135deg, #1a3056 0%, #2c5f8d 50%, #5a8fc4 100%);
   ```

2. **Упростить градиент (более элегантно)**:
   ```css
   background: linear-gradient(135deg, #2c5f8d 0%, #4a7ba7 100%);
   ```

3. **Темный профессиональный стиль**:
   ```css
   background: linear-gradient(135deg, #0f1419 0%, #2c5f8d 50%, #1a3056 100%);
   ```

### Вариант 2: Использовать нейтральные тона

Для более универсального делового стиля:

```css
background: linear-gradient(135deg, #1a1a2e 0%, #16213e 50%, #0f3460 100%);
```

### Вариант 3: Добавить акцентный цвет

Можно добавить **золотой/янтарный акцент** для премиум-ощущения:
- Accent: `#d4af37` (золото) или `#c7a962` (приглушенное золото)

### Primary Colors:
- **Основной синий**: `#2c5f8d` (из логотипа, сохранить)

```css

    #1a3056 0%,    /* Темно-синий */
    #2c5f8d 50%,   /* Основной корпоративный */
    #4a7ba7 100%   /* Светло-синий */
```

### Дополнительные цвета:
- **Текст на светлом**: `#333333`
- **Текст на темном**: `#ffffff`
- **Акцент (кнопки, ховеры)**: `#4a7ba7`
- **Фон карточек**: `#ffffff`
- **Границы**: `rgba(44, 95, 141, 0.1)`

## Применение к видео-фону

Если будет добавлен видео-фон, рекомендуется:

1. **Overlay цвет**: `rgba(26, 48, 86, 0.7)` - темно-синий с прозрачностью
2. **Альтернатива**: `rgba(44, 95, 141, 0.8)` - основной синий

Это обеспечит:
- Читабельность текста поверх видео
- Сохранение корпоративной идентичности
- Гармонию с белым логотипом
